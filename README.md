# ModernTech HR Management System

## Project Description

This project is a Human Resources Management System developed for ModernTech Solutions. The application allows HR staff to manage employees, calculate payroll, process leave requests, and track employee attendance.

## Technologies Used

Vue.js
Bootstrap 5
HTML5
CSS3
JavaScript
Local Storage

## Features

Employee Management
Search Employees
Add and Delete Employees
Payroll Calculations
Digital Payslip Generation
Leave Request Management
Attendance Tracking
Responsive Design

## Installation

Clone the repository.
Open the project folder.
Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
